What we're building.

Tools for operators who run AI agents in fleets — containment, memory, observability, orchestration, and the infrastructure to host it all. Every product runs on the same bare-metal Talos cluster, shares the same identity and secrets plane, and feeds the same data lake.

Agent Infrastructure

v1.0.0

blackwall

AI-containment custody layer. Landlock jail, content-addressed run records, typed settle state machine. k8s orchestration, OTLP telemetry, training corpus export. Nothing lands until you settle it.

 in production

reverie

Memory consolidation daemon for LLM coding harnesses. SQLite + FTS5, MCP server, HTTP API, CLI, TUI. The sole cross-harness memory of record for the estate.

 in development

escapement

Agent-orchestration engine — metered, deterministic dispatch for the agent constellation. Rust core + edge worker. Schedules DAG plans, carries custody provenance through the dispatch lifecycle.

 live

mission-control

Live agent orchestration surface on Cloudflare Workers. Real-time dispatch monitoring, LLM gateway catalog, AgentHub Durable Object, agent lifecycle visualization.

Observability & Tooling

internal

meridian

Read-first cockpit for multi-session agent fleets. Session health, token spend, tool-call traces, settle-state visibility — the command center for fleet operators.

 in development

bench

Model benchmark dashboards with provenance-verified frontier leaderboard. Local content-addressed task bank, reproducible benchmarks, PWA installable.

 v0.1.0

k8s-audit

Kubernetes cluster audit dashboard. SvelteKit PWA for cluster state, GitOps, networking, RBAC, and policy visualization across the Cygnus and Lyra clusters.
