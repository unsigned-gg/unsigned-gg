What we're building.

Tools for operators who run AI agents in fleets and answer for the result. Containment, observability, and the data to prove what happened.

v1.0.0

blackwall

AI-containment custody layer. Runs agents inside a Landlock syscall jail, captures output as content-addressed records, and settles every change behind an explicit decision. k8s orchestration, OTLP telemetry, training corpus export. Nothing lands until you settle it.

 alpha

overflow

Observability surface for agent fleets. Consumes blackwall's REST API and Langfuse data lake to visualize dispatch queues, run custody lineage, and settle-state transitions in real time. The operator's pane for the custody chain.
