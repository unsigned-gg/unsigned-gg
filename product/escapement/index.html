Metered dispatch for the agent constellation.

escapement is the scheduling layer that dispatches blackwall runs as one-shot jobs, manages DAG plan execution, and carries custody provenance through the entire dispatch lifecycle. A Rust core for deterministic ordering, an edge worker for low-latency admission. No agent fires until the escapement allows it.

What it does

  • One-shot dispatch — every blackwall run is a single metered job: admitted, executed, settled, reaped. No long-running agent processes holding state they can't prove.
  • DAG plan execution — multi-step agent plans run as a directed acyclic graph. Dependencies resolved in topological order; each node inherits custody from its predecessors. Mid-run branch points and CRDT overlap merge keep parallel branches consistent.
  • Custody provenance — the dispatch record carries the full chain: who requested the run, what plan node spawned it, which parent custody it descends from, where the output landed. Every job is auditable back to its origin.
  • Metered admission — concurrency limits, rate quotas, priority lanes, and backpressure. The escapement is the choke point that keeps the constellation from overrunning its providers or its budget.

Architecture

Deterministic Rust core for plan resolution and custody state; a lightweight edge worker handles admission, queueing, and provider-side backpressure close to where runs actually fire. The two communicate over a typed dispatch protocol — no ambiguous job JSON crossing the wire.

Why "escapement"

A mechanical escapement is the mechanism that meters a clock's energy into discrete, deterministic ticks — releasing one tooth at a time, never the whole gear train at once. The agent constellation runs the same way: the escapement admits one job per tick, in a known order, with the full custody chain attached. No free-wheeling. No lost beats.

Status

In development. Core dispatch and DAG plan execution are wired against the blackwall v1.0.0 custody API; edge worker admission and metered backpressure are the active work surface. Custody provenance flows end to end but the typed dispatch protocol is still stabilizing before we freeze it.
