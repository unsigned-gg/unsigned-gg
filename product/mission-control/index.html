Every agent, one pane of glass.

mission-control is the live orchestration surface at mission-control.unsigned.gg. A Cloudflare Worker fronts an AgentHub Durable Object; the dashboard watches dispatch, gateway health, and agent lifecycle in real time. The catalog panel is your single map of every LLM gateway on the estate — capacity, cost, latency, failover — at agent.unsigned.gg.

What it does

  • Real-time dispatch monitoring — every run, branch, and settle streamed live from the dispatch queue to the dashboard the second it lands.
  • LLM gateway catalog — one panel for every gateway: model mix, tokens/min, $/Mtok, p95 latency, failover state. No more guessing which gateway to route to.
  • AgentHub Durable Object — a single stateful edge object per workspace holding agent roster, assignment ledger, and heartbeat timeline. Survives Worker evictions, replays in order, never drops a beat.
  • Gateway health — live probes fire on an interval; an unhealthy gateway is fenced off the routing table and surfaced red before the first request fails.
  • Agent lifecycle visualization — spawn → idle → claimed → running → settling → done, rendered as a timeline you can scrub. Branches and escalations are first-class.

Architecture

A thin, globally-distributed edge app. The Worker is the public face — it terminates the dashboard, handles SSE/WebSocket fan-out, and validates the operator session. All durable state lives in the AgentHub Durable Object, so a Worker restart never loses in-flight assignments or the gateway catalog. Gateway probes run on a cron trigger and write their results back through the Object; the dashboard subscribes to a single broadcast stream and re-renders on push.
