The platform that hosts itself.

unsigned-paas is the Kubernetes-native AI/ML platform that runs the entire unsigned ecosystem — including its own control plane. Bare-metal Talos on Vultr, Cilium networking, Longhorn storage, ~27 Helm charts reconciled through ArgoCD GitOps. Every sibling product — blackwall, overflow, reverie — is a workload on this cluster.

What it does

A single self-hosted cluster that takes bare metal and turns it into a developer platform: identities, secrets, registries, CI/CD, observability, and mesh — all declared in Git and reconciled continuously. No managed control plane, no click-ops. The same manifests that stood the cluster up can rebuild it from scratch on a fresh region.

  • Cluster fabric — Talos Linux on Vultr bare metal, immutable K8s API, Cilium eBPF networking, Longhorn distributed block storage
  • GitOps backbone — ~27 Helm charts, ArgoCD ApplicationSets, app-of-apps root, drift detection, prune + self-heal on
  • Identity & secrets — Keycloak OIDC for SSO across every service, OpenBao for dynamic secrets, Tailscale tailnet for operator access
  • Artifact supply chain — Harbor registry with vulnerability scanning, signed image policy, robot accounts per service
  • Observability stack — Prometheus + Grafana dashboards, Loki log aggregation, Jaeger distributed tracing, unified alerting via Alertmanager
  • CI/CD pipelines — container builds push to Harbor, ArgoCD auto-syncs on image-tag bump, rollback is a Git revert
  • SOPs & runbooks — every operational action documented as a procedure: swaps, rotations, incident response, cluster rebuild

Why bare-metal Talos

No managed control plane means no API bills, no vendor lock-in, and full control over the kernel surface — which matters for blackwall's Landlock jail. Talos gives an immutable, API-driven node image with zero host SSH; everything flows through the K8s API or the Tailscale sidechannel. Vultr bare metal keeps latency to the GPU nodes low and the bill predictable.
