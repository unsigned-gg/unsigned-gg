0x01

What it does.

Every feature is built for developers who live in the terminal and run AI agents alongside it. No cloud, no accounts, no telemetry nags.

/01
shipped

GPU-accelerated rendering

wgpu backend — D3D12 on Windows, Vulkan on Linux, Metal on macOS. One API, three platforms. Glyph atlas with instance-based text rendering, damage tracking, 60fps target. No OpenGL legacy, no CPU-bound rendering.

/02
shipped

Full VT/ANSI parser

vte 0.15 with complete Perform trait. Scroll regions, terminal modes, tab stops, ESC sequences, synchronized output, styled underlines (curly/dotted/dashed), mouse reporting, Kitty keyboard protocol, grapheme clusters, ligature shaping. 77 integration tests against vim, htop, less, tmux.

/03
shipped

Shell integration & blocks

OSC 133 semantic prompt parsing — command/output blocks with exit codes, duration, and timestamps. Block navigation (Ctrl+Shift+Up/Down). Command history (bounded, searchable). Shell scripts for bash, zsh, fish. Like Warp, but local-only: no cloud, no AI, no account.

/04
shipped

Image protocols

Sixel (DCS q), iTerm2 inline images (OSC 1337), Kitty Graphics Protocol (APC pre-scan). Image store with RGBA data and cell placements. WGSL shader for textured quads. DCS hook/put/unhook callbacks. Behind --features images.

/05
shipped

Command palette & quake mode

Ctrl+Shift+P opens a fuzzy-filtered command palette — 17 commands, arrow-key navigation. Ctrl+Grave toggles quake/drop-down mode: borderless, top-half-of-screen, full-width. 7 built-in themes (unsigned, dim, high-contrast, solarized-dark/light, synthwave84, oni).

/06
shipped

Domain model extraction

overflow-domain crate — 28 modules of pure data types (no I/O, no parsing, no rendering). 2-crate Cargo workspace. PGT type schemas for cross-project sharing with godseat. Foundation for mux, remote, demon mode, and attention router.

/07
building

Agent-aware mux (herdr)

Tabs, panes, split/resize. Named sessions with detach/reattach. Agent status bar: blocked / working / done / idle per pane. Detects Claude Code, Codex, Amp, OpenCode via process name or OSC 133 markers.

/08
planned

Mosh remote (native Windows)

No native Mosh client exists on Windows. Overflow speaks the protocol natively — built-in mosh-server (connect from iPhone/Blink) and mosh-client. UDP state sync, predictive local echo, roaming that survives IP changes and sleep/wake.

/09
planned

Demon mode (operator control surface)

Ctrl+Alt+D transforms the terminal into a cluster dashboard. godseat integration: estate sensing, gate classification (Green/Yellow/Red/Never), sign-off prompts, audit trail. Read-only dormancy mode. K8s CRD awareness. No terminal in the landscape has this.

/10
planned

Attention router

When N parallel agent sessions run, one blocks on you — needs permission, has a prompt ready. The router auto-promotes the blocked pane to the driver slot. Engage it, the next waiter surfaces. Drain the queue hands-free without touching the layout.

/11
planned

Reverie memory + Somnium agents

Session snapshots persist to engram — scrollback, block history, agent state survive reboots. Somnium: persistent per-project agents with canon gating and preview deploys. Block history as change timeline. The terminal IS the audit view.

0x03

Roadmap to v1.0.0.

11 phases, each a version bump. 6 shipped, 5 ahead. No dates we can't keep.

v0.1.0-alpha MVP terminal Type a command, see colored output. VT/ANSI parser, wgpu surface, async PTY, config, telemetry. shipped
v0.2.0-alpha Alacritty parity Full VT/ANSI parser: scroll regions, modes, tab stops, styled underlines, mouse, keyboard, graphemes, ligatures, damage tracking, 77 integration tests. shipped
v0.3.0-alpha herdr mux Agent-aware mux: tabs, panes, split/resize, detach/reattach, agent status bar (blocked/working/done/idle). building
v0.4.0-alpha Blocks OSC 133 semantic prompts, block detection, command history, block navigation, shell integration scripts. shipped
v0.5.0-alpha Images Sixel, iTerm2 inline images, Kitty Graphics Protocol. Image store, WGSL shader, DCS callbacks. shipped
v0.6.0-alpha Mosh remote Native Windows mosh server/client. UDP state sync, predictive local echo, roaming. Connect from iPhone/Blink. planned
v0.7.0-alpha Demon mode godseat integration: estate sensing, gate classification, sign-off prompts, audit trail, dormancy mode, K8s awareness. planned
v0.8.0-alpha Attention router Priority queue: blocked panes auto-promote to driver slot. Hold/resume, window isolation, herdr status as signal. planned
v0.9.0-alpha Reverie memory Session snapshots to engram. Block metadata persistence. Session restoration on attach. Estate sensing for demon mode. planned
v0.10.0-alpha Somnium Persistent per-project agents. Canon-change gating. Preview deployment on agent commit. Block history as change timeline. planned
v1.0.0 Polish Command palette, quake mode, theme system, config hot-reload, binary size optimization, Windows installer. Feature-complete. in progress
